# camera.ui Go SDK

Go SDK for building plugins that extend [camera.ui](https://github.com/seydx/camera.ui) — add cameras, run motion / object / face / license-plate / audio detection, deliver notifications, and more.

## Install

```bash
go get github.com/cameraui/sdk/go
```
